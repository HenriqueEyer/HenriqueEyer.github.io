Detalhes

Proposito do programa:

  1. Um programa para ajudar um professor do curso Engenharia de Minas, da faculdade UFOP. O programa servirá para ajudar na materia de geo processamento, usar como base um programa já existente mas melhora lo dando novas funções como ao adicionar uma amostra ele já mudar automaticamente o valor final, sem a necessidade de recolocar os valores de novo para receber resultado final, e torna lo mais interativo com o usuario.
  2. Ajudar os alunos, ao acesso ao programa e facilitar seu aprendizado.
  3. Testar meus conhecimentos de Programação, e coloca los em pratica.
